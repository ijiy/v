# TypeScript 【介绍】

## 关于
* TypeScript是在JavaScript的基础上添加静态类型定义构建而成的编程语言

* TypeScript开发的最终结果还是JavaScript, 这个结果由TypeScript编译器或Babel转译而成

## 特点
* 基于类`class`的面向对象编程

* IDE类型检查和高亮支持

* webpack, vite等框架的支持

* uniapp的uts, egret的ts等,
