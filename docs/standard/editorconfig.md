# IDE编辑配置 【.editorconfig】

[.editorconfig](https://github.com/editorconfig/editorconfig/wiki/EditorConfig-Properties)

```bash
root = true

[*.{js,ts,vue,nvue,scss,md,json,html}]

indent_style = tab
indent_size = 4

charset = utf-8

end_of_line = crlf

trim_trailing_whitespace = true

insert_final_newline = true
```
